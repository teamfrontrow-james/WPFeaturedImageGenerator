# Support

For setup and usage help, start with:

- [README.md](README.md)
- [API setup](docs/api-setup.md)
- [Troubleshooting](docs/troubleshooting.md)
- [Security notes](docs/security.md)

When opening a support issue, include:

- WordPress version.
- PHP version.
- Plugin version.
- Whether the site is public, staging, or local.
- The exact dashboard status or error message.
- Relevant debug-log lines with API keys and private content redacted.

Do not post API keys, raw provider responses containing secrets, unpublished customer content, or private generated image URLs in public issues.
